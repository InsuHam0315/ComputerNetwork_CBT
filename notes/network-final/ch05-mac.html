1. 챕터 개요

LAN 환경에서 매체 접근과 프레임 전달을 담당하는 MAC 계층을 시험 대비 관점으로 정리한다.

2. 시험에 나오는 핵심

  • IEEE 802 계열과 LAN 표준의 관계
  • MAC과 LLC의 역할 구분
  • CSMA/CD의 감지-충돌-백오프 흐름
  • 토큰 버스와 토큰 링의 권한 기반 송신
  • 프레임 구조와 오류 검출

3. 핵심 개념 설명

  • LAN/MAN 표준을 체계화한 IEEE 표준 계열
  • 매체 접근 제어와 물리 주소 기반 프레임 전달 기능
  • 상위 계층에 논리 링크 서비스를 제공하는 하위 계층
  • 송신 전 매체 감지와 충돌 감지를 사용하는 경쟁 기반 방식
  • 토큰 버스와 토큰 링에서 송신 권한을 나타내는 제어 정보
  • 프레임 오류 검출을 위해 붙는 검사 정보
  • 링 형태 논리 구조에서 토큰으로 송신 순서를 제어하는 방식
  • MAC 주소를 학습해 필요한 포트로 프레임을 전달하는 장비
