1. 챕터 개요

IPv6, 이동 IP, 터널링, ARP/RARP, ICMP, IGMP를 주소 변환과 제어 메시지 관점에서 정리한다.

2. 시험에 나오는 핵심

  • IPv6 주소 길이와 헤더 구조
  • 이동 IP와 터널링의 필요성
  • ARP/RARP 주소 변환 방향
  • ICMP 오류·진단 메시지
  • IGMP 멀티캐스트 그룹 관리

3. 핵심 개념 설명

  • 128비트 주소와 확장 헤더 구조를 사용하는 IP 버전
  • 기본 헤더와 확장 헤더로 기능을 분리할 수 있는 구조
  • 한 프로토콜 패킷을 다른 패킷 안에 캡슐화해 전달하는 기법
  • IP 주소를 이용해 MAC 주소를 찾는 프로토콜
  • MAC 주소를 이용해 IP 주소를 찾는 프로토콜
  • IP 오류 보고와 진단성 제어 메시지를 제공하는 프로토콜
  • 멀티캐스트 그룹 관리를 위한 프로토콜
  • 노드가 네트워크를 이동해도 통신 지속성을 지원하는 개념

5. 계층별 기능 정리

  • IPv6, 이동 IP, 터널링, ARP/RARP, ICMP, IGMP를 주소 변환과 제어 메시지 관점에서 정리한다.
  • 하위 계층의 전달 기능을 상위 계층 서비스와 연결해서 이해합니다.
