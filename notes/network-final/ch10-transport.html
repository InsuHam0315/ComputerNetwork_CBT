1. 챕터 개요

UDP, RTP, 세션 프리미티브, OSI TP 프로토콜을 전송 계층 서비스 관점에서 정리한다.

2. 시험에 나오는 핵심

  • UDP의 비연결형·단순 헤더
  • UDP와 TCP 비교
  • RTP의 순서 번호와 타임스탬프
  • RTP 릴레이와 실시간 전달
  • 세션 프리미티브와 OSI TP

3. 핵심 개념 설명

  • 비연결형이며 헤더가 단순한 전송 계층 프로토콜
  • 출발지 포트, 목적지 포트, 길이, 체크섬 중심의 단순 헤더
  • 실시간 오디오·비디오 전송에서 순서와 시간 정보를 보조하는 프로토콜
  • 실시간 미디어 흐름을 중계하거나 전달 경로를 보조하는 기능
  • 순서 번호와 타임스탬프 등 실시간 전송 정보를 담는 헤더
  • 연결 설정, 데이터 전송, 연결 해제 같은 서비스 기본 동작
  • 전송 계층에서 응용 프로세스를 구분하기 위한 번호
  • OSI 관점의 전송 프로토콜 계열
