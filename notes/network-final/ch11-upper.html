1. 챕터 개요

세션·표현·응용 계층의 역할과 토큰, 동기, 데이터 표현, 클라이언트-서버 모델을 정리한다.

2. 시험에 나오는 핵심

  • 세션 계층의 대화 제어·토큰·동기
  • 표현 계층의 데이터 표현·압축·암호화
  • 응용 계층의 사용자 서비스
  • 클라이언트-서버 모델
  • 상위 계층 기능 비교

3. 핵심 개념 설명

  • 대화 제어, 토큰 관리, 동기점 설정을 담당하는 계층
  • 세션에서 특정 동작 수행 권한을 제어하는 논리 정보
  • 오류 발생 시 복구 기준점으로 사용할 수 있는 표시
  • 데이터 표현 형식, 압축, 암호화 변환을 담당하는 계층
  • 전송하거나 저장할 데이터 양을 줄이는 기능
  • 사용자에게 네트워크 응용 서비스를 제공하는 계층
  • 서비스 요청자와 제공자를 구분하는 응용 구조
  • 시스템 간 자료 형식 차이를 맞추는 기능

4. 주요 프로토콜/용어 정리

세션 계층대화 제어, 토큰 관리, 동기점 설정을 담당하는 계층
토큰세션에서 특정 동작 수행 권한을 제어하는 논리 정보
동기점오류 발생 시 복구 기준점으로 사용할 수 있는 표시
표현 계층데이터 표현 형식, 압축, 암호화 변환을 담당하는 계층
데이터 압축전송하거나 저장할 데이터 양을 줄이는 기능
응용 계층사용자에게 네트워크 응용 서비스를 제공하는 계층
클라이언트-서버 모델서비스 요청자와 제공자를 구분하는 응용 구조
데이터 표현시스템 간 자료 형식 차이를 맞추는 기능

5. 계층별 기능 정리

  • 세션·표현·응용 계층의 역할과 토큰, 동기, 데이터 표현, 클라이언트-서버 모델을 정리한다.
  • 하위 계층의 전달 기능을 상위 계층 서비스와 연결해서 이해합니다.

6. 헷갈리는 비교표

세션 계층대화·동기·토큰표현 계층형식·압축·암호화
표현 계층데이터 표현 변환응용 계층사용자 서비스 제공
클라이언트서비스 요청서버서비스 제공

7. 교수님 스타일 빈칸 포인트

세션 계층토큰동기점표현 계층데이터 압축응용 계층클라이언트-서버 모델데이터 표현

8. 모두 고르기 예상 선지

  • 세션 계층 기능으로 옳은 것을 모두 고르시오.
  • 표현 계층 기능으로 옳은 것을 모두 고르시오.
  • 응용 계층에 대한 설명으로 옳은 것을 모두 고르시오.
  • 클라이언트-서버 모델 설명으로 옳은 것을 모두 고르시오.

9. 설명형 예상문제

  • 세션 계층에서 동기점의 목적을 설명하시오.
  • 표현 계층의 주요 기능을 설명하시오.
  • 클라이언트-서버 모델을 설명하시오.

10. 최종 암기 요약

  • 세션 계층의 대화 제어·토큰·동기
