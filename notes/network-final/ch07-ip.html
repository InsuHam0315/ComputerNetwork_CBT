1. 챕터 개요

네트워크 계층의 라우팅, 혼잡 제어, IP 헤더, 분할, DHCP를 핵심 중심으로 정리한다.

2. 시험에 나오는 핵심

  • 네트워크 계층의 종단 간 패킷 전달
  • 연결형/비연결형 서비스 비교
  • 라우팅과 혼잡 제어 구분
  • 거리 벡터/링크 상태/외부 라우팅
  • IP 헤더와 패킷 분할, DHCP

3. 핵심 개념 설명

  • 여러 네트워크를 거쳐 목적지까지 패킷 전달과 경로 선택을 담당하는 계층
  • 목적지까지의 전달 경로를 선택하는 기능
  • 네트워크 과부하를 완화하기 위해 전송량을 조절하는 기능
  • 이웃과 목적지까지의 거리 정보를 교환하는 라우팅 방식
  • 링크 정보를 수집해 토폴로지 기반으로 경로를 계산하는 방식
  • 자율 시스템 사이의 경로 선택과 관련된 라우팅
  • IP 패킷 전달과 분할에 필요한 주소·제어 정보를 담는 부분
  • 호스트에게 IP 주소와 네트워크 설정을 자동 제공하는 프로토콜

4. 주요 프로토콜/용어 정리

네트워크 계층여러 네트워크를 거쳐 목적지까지 패킷 전달과 경로 선택을 담당하는 계층
라우팅목적지까지의 전달 경로를 선택하는 기능
혼잡 제어네트워크 과부하를 완화하기 위해 전송량을 조절하는 기능
거리 벡터이웃과 목적지까지의 거리 정보를 교환하는 라우팅 방식
링크 상태링크 정보를 수집해 토폴로지 기반으로 경로를 계산하는 방식
외부 라우팅자율 시스템 사이의 경로 선택과 관련된 라우팅
IP 헤더IP 패킷 전달과 분할에 필요한 주소·제어 정보를 담는 부분
DHCP호스트에게 IP 주소와 네트워크 설정을 자동 제공하는 프로토콜

5. 계층별 기능 정리

  • 네트워크 계층의 라우팅, 혼잡 제어, IP 헤더, 분할, DHCP를 핵심 중심으로 정리한다.
  • 하위 계층의 전달 기능을 상위 계층 서비스와 연결해서 이해합니다.

6. 헷갈리는 비교표

연결형연결 설정 후 상태 기반 전달비연결형패킷별 독립 전달
거리 벡터이웃 중심 정보 교환링크 상태토폴로지 기반 계산
흐름 제어수신자 처리 능력혼잡 제어네트워크 과부하

7. 교수님 스타일 빈칸 포인트

네트워크 계층라우팅혼잡 제어거리 벡터링크 상태외부 라우팅IP 헤더DHCP

8. 모두 고르기 예상 선지

  • 라우팅에 대한 설명으로 옳은 것을 모두 고르시오.
  • 거리 벡터 라우팅의 특징으로 옳은 것을 모두 고르시오.
  • 링크 상태 라우팅의 특징으로 옳은 것을 모두 고르시오.
  • DHCP에 대한 설명으로 옳은 것을 모두 고르시오.

9. 설명형 예상문제

  • 거리 벡터와 링크 상태 라우팅을 비교하시오.
  • IP 패킷 분할이 필요한 이유를 설명하시오.
  • DHCP가 제공하는 기능을 설명하시오.

10. 최종 암기 요약

  • 연결형/비연결형 서비스 비교
  • 라우팅과 혼잡 제어 구분
