1. 챕터 개요

TCP의 연결 설정·해제, 헤더 필드, 데이터 전송, 흐름/오류/혼잡 제어를 정리한다.

2. 시험에 나오는 핵심

  • TCP의 연결형·신뢰성 서비스
  • TCP 헤더의 포트, 순서 번호, ACK 번호
  • 3-way handshake와 연결 해제
  • 흐름 제어와 오류 제어
  • 혼잡 제어와 송신량 조절

3. 핵심 개념 설명

  • 연결형 신뢰성 전송 계층 프로토콜
  • TCP가 데이터를 나누어 전송할 때 사용하는 단위
  • SYN, SYN+ACK, ACK 흐름으로 연결을 설정하는 절차
  • TCP 연결 설정 시작을 나타내는 제어 플래그
  • TCP 연결 종료 요청과 관련된 제어 플래그
  • 수신 윈도우를 고려해 송신량을 조절하는 기능
  • ACK, 체크섬, 재전송으로 손실·오류를 복구하는 기능
  • 네트워크 혼잡 상태를 고려해 송신 속도를 조절하는 기능
