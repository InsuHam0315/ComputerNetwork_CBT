Chapter 05 MAC 계층

IEEE 802, LAN, MAC/LLC, CSMA/CD, 토큰 방식, 프레임 오류

MAC 계층은 같은 전송 매체를 여러 장치가 공유할 때 누가 언제 프레임을 보낼지 정하는 부분이다. 시험에서는 MAC을 단순히 주소 이름으로만 보지 말고, 매체 접근 제어와 프레임 전달 규칙을 함께 묶어서 이해해야 한다.

Chapter 06 데이터 링크 계층

프레임, 흐름 제어, 오류 제어, 슬라이딩 윈도우, HDLC/LAP

데이터 링크 계층은 직접 연결된 인접 노드 사이에서 프레임 단위 전송을 관리한다. 물리 계층이 비트를 보내는 데 집중한다면, 데이터 링크 계층은 그 비트를 프레임으로 묶고 오류와 속도 차이를 제어한다.

Chapter 07 IP 프로토콜

네트워크 계층, 라우팅, 혼잡 제어, 라우팅 프로토콜, IP 헤더, 분할, DHCP
